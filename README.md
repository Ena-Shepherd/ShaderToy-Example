## How to use
- **Linux** :
  - `./shader-demo <relative_.glsl_path>`<br/><br/>
- **Windows** :
  - Right-click **shader-demo.exe**
  - new Shortcut 
  - Properties
  - target : `<shader-demo.exe_current_location> <relative_.glsl_path>`
